How Do I Control My VISA Device from Another Computer?

Primary Software: Driver Software>>NI-VISA
Primary Software Version: 1.1
Primary Software Fixed Version: N/A
Secondary Software: N/A

Problem:
I want to control a VISA device from a remote computer. How can I do this?

Solution:
Note: You must have NI-VISA Run-Time installed in order to use the NI-VISA server and client.  Go to ni.com/drivers and search for "Visa Run-Time" to find the latest and previous version of this application.

Server Setup:

1. Open Measurement & Automation Explorer and navigate to Tools » NI-VISA » VISA Options.  You should see this selection tree. Select VISA Server, it is marked by the red box in the image.



2.  In the server options that is now open, if you want the VISA server to run on startup you should enable the checkbox located in the red box in the image below.  Now press the Start Server now button, shown in the blue box in the image below.



3.  If you are running Windows 7, you will probably recieve this message.  Make sure to allow the server to access your network by pressing the Allow Access button. You can also enable the check boxes if you want this computer to allow access to the server over public or private networks.



4. Now you need to configure access to your VISA server.  Select the Security sub-option from the left hand menu, Security is located under the VISA Server option.  If you want to allow access to this computer's VISA server from any remote computer you can add a wildcard '*' to the remote access list.  The picture below shows a VISA server configured to allow access from any remote computer.  The permission selector that is greyed out is set to "Access only" for the wildcard entry. 

You can also add remote machines by IP address or DNS Name



5. Once you have configured your remote access list.  Then click the save button in the upper left hand part of the window. Your computer is now configured to share its VISA resources at the address visa://<machine name>/<resource name>.



Client Setup:

1.  In order to use the VISA client you must have the VISA run-time engine installed.

2. After you have verified you have the necessary software, launch Measurement & Automation Explorer.

3. Open the VISA Interactive Control window by going to Tools » NI-VISA » VISA Interactive Control.  Below is a screenshot of the window that will pop up.



4. In the "Machine:" text box enter the name or IP address of your VISA server, or you can use the drop down to target a machine on your local network that MAX has already found.  The list of resources should refresh, and you can double click a resource to bring up a test panel window.  The address of the resource can be used by VISA methods to open and control the remote resource, as shown below.
